Fan Control System Simulation

Overview
The Fan Control System Simulation is a Python GUI application that simulates a fan remote controller 
using Tkinter and Object-Oriented Programming (OOP). The program allows users to control fan speed levels and 
display animated GIF simulations based on the selected speed.The project applies concepts such as encapsulation, 
classes, objects, threading, and GUI event handling.

Libraries 
- Tkinter (tkinter)
OS
- os
Threading
- threading

Features
- GUI fan remote controller
- Slow, Medium, and Fast fan speed modes
- Power OFF functionality
- Animated GIF playback based on fan speed
- Dynamic speed display
- OOP-based program structure
- Multi-threaded GIF loading for smoother performance
