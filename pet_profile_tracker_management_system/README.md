Pet Profile Tracker Management System

Overview
The Pet Profile Tracker Management System is a Python GUI application that allows users to register pets, 
upload optional pet photos, and generate a visual pet ID card containing the pet's information.The program applies 
Object-Oriented Programming (OOP) concepts such as encapsulation, classes, objects, getters, and setters.

Libraries Used
Tkinter
    - tkinter
    - filedialog
    - messagebox
Pillow (PIL)
    - Image
    - ImageTk
Random

Features
- Register pet information
- Generate automatic pet ID numbers
- Upload pet profile images
- Generate visual pet ID cards
- Input validation and error handling
- GUI-based interaction using Tkinter
