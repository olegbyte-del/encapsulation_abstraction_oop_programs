Car Motion Control System Simulation
A Tkinter-based car controller simulation that changes speed using GAS and BRAKE buttons and displays 
different GIF animations based on the car’s current speed. The project focuses on Object-Oriented Programming, 
GUI development, and simple animation handling in Python.

Features
- Car speed control (accelerate and brake)
- GUI-based controller using Tkinter
- GIF animation changes based on speed
- Real-time speed display updates
- Brake system with speed reduction logic
- Separate animation window using Toplevel

Libraries Used
- tkinter 
- Pillow (PIL) 
- os 
- threading
